SjlSet

new SjlSet(iterable)

SjlSet constructor. This object has the same interface as the es6 `Set` object. The only difference is this one uses a more sugery iterator which has, in addition to the `next` method, `current`, `iterator`, `pointer`, `rewind`, and `valid` methods (@see sjl.Iterator)
