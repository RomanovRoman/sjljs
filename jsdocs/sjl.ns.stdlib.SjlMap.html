Class: SjlMap

SjlMap

new SjlMap(iterable)

'Simple Javascript Library Map' object (stand-in object for es6 Maps until they're support is more widely accepted).

This constructor offers the same exact api as es6 Map objects with an additional couple of convenience methods (addFromArray, addFromObject, iterator, toJson).

Name Type Description
iterable Array | Object

The object to populate itself from (either an Array<[[key, value]]> or an Object hash).
